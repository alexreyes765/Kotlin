<h2>Funcionalidad acorde al caso</h2>

<p>El caso práctico elegido fue una calculadora de índice de masa corporal, te pide ingresar los datos como lo son: la estatura, el peso y la edad, cuenta con un botón que al momento de dar click te pide permisos para poder mandarte notificaciones,</p> 
<p>en la barra de notificaciones te mostrara el IMC correspondiente, también cuenta con un menú donde además de ver dos botones: calculadora de IMC y otro que se llama Saludapp, Esta te pide que ingreses tu nombre y te redirige a otra pantalla donde</p>
<p>te muestra solo un saludo con el nombre ingresado y un gif de un erizo bailando, en la parte de abajo cuenta con un botón para regresar a la pantalla anterior, para salir de ahí hay que deslizar la pantalla de izquierda a derecha, por ultimo cuenta</p> 
<p></p>con una pantalla de inicio(splash screen) que muestra una imagen, esto ayuda a que la app al momento de arrancar no se vea solo una pantalla en blanco.</p>
